# 📝 Simple To-Do List

## ✨ Features

* **Add Tasks:** Quick input field to append new items to your list.
* **Complete Tasks:** Toggle checkboxes or click items to mark them as done.
* **Delete Tasks:** Remove individual tasks instantly with a delete/trash button.
* **Responsive Design:** Fully optimized for desktop, tablet, and mobile screens.

## 🛠️ Technologies Used

* **HTML5:** Structured the layout and core text input elements.
* **CSS3:** Custom styling, smooth hover transitions, and a clean user interface.
* **JavaScript :** Handled DOM manipulation and task logic arrays.

## 📂 Project Structure

├── index.html       # Main HTML structural file
├── style.css        # Custom CSS styling rules
└── script.js        # Core JavaScript application logic
